Lazy worker:

TypoRoll

Typo Roll is a non-digital machine that can write any word of maximum five letters. It is the result of a one week collaboration with Léonard Guyot and Maya Bellier.

We created a modular typography composed of three modules, each modules are used in minimum two different characters. Then we built this wooden sculpture with which we can interact and write our desired words.
